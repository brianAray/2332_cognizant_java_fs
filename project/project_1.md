# Project 1

This is a team project (Groups of 3 - 4) and you will be required to present what you have done.

- Due date: 1/28/26

- Technology Requirements:
	- Version Control: Git
	- Frontend: Angular or React
	- Backend: Spring Boot
	- Authentication: JWT
	- Database: PostgreSQL or MongoDB
	- Testing:
		- API: Postman
		- Unit: JUnit5 and Mockito
		- Frontend: Jasmine and Karma / React Testing Library and Jest
- Feature Requirements:
	- Frontend must use a frontend framework
	- Persistent data
	- Backend API must be serving and receiving JSON
	- Some form of login and registration
	- API must have protected endpoints, and JWT must be used for authentication
	- There should be at minimum 4 types of users: Guest, Basic Users, Admin Users, and Super User
- Project Requirements:
	- Before project development can begin, a project proposal must be presented and approved by me
	- Project must follow Agile and Scrum, required team roles: Notetaker, Scrum Lead, and Team Lead
	- Team standups must be conducted everyday and notes must be saved and tracked
	- Project development must be on a Kanban Board (i.e. Trello)
	- Project must be presented with a demo and presentation slides
